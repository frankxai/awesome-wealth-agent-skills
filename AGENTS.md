## Agent Rules

- Read this file before making changes.

## Public Boundary

Allowed:

- Public GitHub repo links and metadata.

- Non-advisory system architecture.

Not allowed:

- Private portfolio details.
- Wallet addresses connected to real holdings.
- Seed phrases, keys, recovery details, or custody locations.
- Live trading strategy, return promises, or capital allocation instructions.
- Customer, founder, LP, deal, or internal strategy data.
